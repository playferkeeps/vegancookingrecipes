## Best Practices

### Posting Frequency

- Instagram: 1-2 times per day
- Facebook: 1 time per day
- X (Twitter): 2-3 times per day

### Optimal Times (Platform-Dependent)

- Instagram: 11 AM - 1 PM, 7-9 PM
- Facebook: 1-3 PM, 7-9 PM
- X: 8-10 AM, 12-1 PM, 5-6 PM

### Content Variety

- Rotate between different recipe categories
- Vary engagement questions
- Mix up hashtag combinations
- Use different emoji combinations

## Monitoring Engagement

Track these metrics:

- **Comments**: Are people responding to questions?
- **Saves/Shares**: Are people saving recipes?
- **Link Clicks**: Are people visiting the site?
- **Reach**: Is content reaching new audiences?

## Usage

The enhanced engagement features are automatically included in all posts. No additional configuration needed - just run the scripts as usual:

```bash
npm run post-recipe-to-instagram
npm run post-recipe-to-facebook
npm run post-recipe-to-x
npm run post-to-social-media  # Posts to all platforms
```

The scripts will automatically:

- Select random engagement questions
- Choose appropriate recipe hooks
- Generate optimized hashtags
- Create engaging captions
